# ⚓ ChatAnchor

ChatAnchor ist eine Chrome-Erweiterung, mit der du wichtige Textstellen in KI-Chats (Gemini, ChatGPT, Claude) markieren und per Klick sofort wieder dorthin zurückspringen kannst.

## Features
- **Rechtsklick-Anker:** Markiere Text und speichere ihn direkt über das Kontextmenü.
- **Smart-Scroll:** Das Plugin findet den Text auch in langen Verläufen durch automatisches Hochscrollen.
- **Bestätigungs-Feedback:** Sofortige Rückmeldung auf der Seite beim Speichern.

## Installation
1. Lade diesen Ordner herunter.
2. Öffne `chrome://extensions/` in Chrome.
3. Aktiviere den **Entwicklermodus**.
4. Klicke auf **Entpackte Erweiterung laden** und wähle diesen Ordner aus.
